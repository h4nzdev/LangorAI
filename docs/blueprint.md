# **App Name**: Langor AI

## Core Features:

- AI Voice Conversation: Engage in natural, real-time English conversations with an AI tutor through voice input and output, providing a judgment-free practice environment.
- AI-Powered Speech Analysis: Receive a tool for comprehensive feedback on fluency, grammar, pronunciation, and vocabulary after each conversation session.
- Progress Tracking & Visualization: Monitor and visualize your English speaking improvement over time through tracked metrics such as confidence level, vocabulary growth, and accuracy. This data will be stored in a database (like MongoDB or SQLite).
- Personalized Learning Recommendations: Get tailored practice strategies and learning recommendations generated by AI based on your past performance and identified areas for improvement.
- Learning Video Library: Access a curated collection of educational videos covering pronunciation techniques, common English mistakes, and grammar lessons for supplementary learning.
- BYO API Key Integration: Integrate your own AI API keys to reduce operational costs and customize your experience, suitable for developers or institutions.

## Style Guidelines:

- A calm, professional blue (#2E73B8) as the primary color, symbolizing confidence and clarity in communication for the light color scheme.
- A very light, desaturated blue-grey (#F0F5F8) for the background, ensuring readability and an inviting, clean feel.
- A vibrant aqua (#3EDFE3) for an accent color, providing contrast and highlighting key interactive elements, reflecting a dynamic learning process.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-forward, and crisp feel. Body text font: 'Inter' (sans-serif) for excellent readability in longer feedback and reports.
- Use clean, contemporary, and easily understandable icons to represent language learning concepts, progress, and feedback elements, enhancing clarity and user experience.
- Implement a structured and intuitive layout, prioritizing easy access to conversation features, progress reports, and learning resources to ensure a seamless learning journey.
- Incorporate subtle and smooth animations for feedback presentation, progress visualizations, and navigation transitions to make the learning experience more engaging and fluid.
